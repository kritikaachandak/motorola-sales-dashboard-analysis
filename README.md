# Motorola Sales Dashboard Analysis | Power BI & Excel Project

## Project Overview
Developed an interactive Power BI dashboard to analyze Motorola mobile sales performance, customer transactions, payment methods, and sales trends across different cities and months.

---

## Objectives
- Analyze total sales and transaction trends
- Compare mobile model performance
- Study customer ratings and payment methods
- Visualize city-wise sales distribution
- Generate business insights using interactive dashboards

---

## Tools & Technologies Used
- Power BI
- Microsoft Excel
- Data Visualization
- Data Cleaning

---

## Dashboard Features

### Sales KPIs
- Total Sales
- Total Quantity
- Transactions
- Average Sales

### Analysis Included
- City-wise sales analysis
- Monthly quantity trends
- Customer ratings analysis
- Payment method distribution
- Mobile model performance
- Day-wise sales trends

### Interactive Features
- Dynamic filters and slicers
- Multi-visual dashboard
- KPI cards and charts
- Interactive map visualization

---

## Key Insights
- Identified top-performing mobile models
- Analyzed customer transaction behavior
- Compared payment preferences
- Evaluated city-wise sales performance
- Tracked monthly and weekly sales trends
